# Minimalist Brand Website Template
        
hjälp mig skapa en simpel hemsida för mitt varumärke. 
startsidan ska innehålla en header med navbar, hero image, bild carousell, signup formulär & footer. 
navbaren (från vänster till höger), längst till vänster ska loggan vara. längst till höger ska det för tillfället finnas: PROJECTS, ABOUT och CONTACT (innan uppdatering då produkter och annat ska finnas, då kommer även CART och SEARCH finnas).
under navbaren ska det vara en stor bild (hero image) som täcker upp hela skärmen. denna bild ska länkas till PROJECTS-sidan. 
med ett litet mellanrum under hero image så ska bild carosellen ligga. carousellen ska funka på ett sätt att man scrollar/shift scrollar åt höger och vänster för att scrolla bland bilderna. det ska alltså vara en bildremsa med bild 1 som start och bild X som sista bild på denna remsa där det tar stopp. bilderna i remsan ska ha ett väldigt litet mellanrum. 
alla bilder och knappar och rutor ska ha 0 i boarder radius, alltså kantiga kanter, inte rundade kanter. fonten på alla text ska vara Helvetica Neue, Regular. All caps.
under bild remsan så kommer footern. 
i footern längst ner till vänster ska det finnas texten NEWSLETTER. när man klickar på newsletter så ska ett signup formulär komma ut åt höger från newsletter texten. formuläret ska vara en tunn input ruta med label text "EMAIL ADRESS" och bredvid input rutan en knapp som är lika tunn där det står "SUBSCRIBE".
längst till höger i footern ska det stå "© 2025 SOT".

PROJECTS-sidan: 
samma header med navbar.
denna sida ska innehålla en samling av olika projekt. uteseendemässigt så ska det vara bilder med en kort beskrivande text under bilden. projekten (bilderna) ska ligga på samma sätt som bild remsan och man scrollar höger och vänster för att se flera projekt.  
samma footer.

ABOUT-sidan:
samma header.
denna sida ska endast innehålla en rubrik "ABOUT" som ligger i mitten till vänster.
i mitten till höger så ska about texten ligga. 
samma footer.

CONTACT-sidan:
samma header.
samma uppbyggnad som about-sidan. till vänster "CONTACT" och till höger alla kontakt uppgifter, som mail och sociala medier. (contact@sotstockholm.se, @sotstockholm (instagram))

denna hemsida ska byggas med next.js, react, typescript, tailwind css. 
bygg den så smart som möjligt för när jag ska lägga till produkt sida, betalningslösning, kassa och checkout.

## Getting Started

Install dependencies and start the development server:

```bash
npm install
npm run dev
```

For production, build and start the server:

```bash
npm run build
npm run server
```
